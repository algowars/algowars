# Backend Setup

Follow these steps to get the backend of **Algowars** up and running.

### Prerequisites

- **Node.js** (v16+)
- **PostgreSQL** database
- **NestJS CLI**

### Steps

1. **Install Dependencies**:
   ```bash
   cd backend
   npm ci
   ```
2. **Setup Database**:
   Setup database and populate the env variables to connect that database.
