# Client Setup

Follow these steps to get the backend of **Algowars** up and running.

### Prerequisites

- **Node.js** (v16+)
- **PostgreSQL** database
- **NestJS CLI**

### Steps

1. **Install Dependencies**:
   ```bash
   cd client
   yarn install
   ```

Now you can use the scripts passed in the package.json file to run/test/generate the application.
